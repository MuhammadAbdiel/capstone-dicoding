# GoWisata

Aplikasi ini dapat diakses [disini](https://gowisata.netlify.app/)
<br>
<br>

# About

GoWisata adalah aplikasi pembelian tiket online yang dapat membantu anda dalam membeli tiket wisata. Aplikasi ini memiliki beberapa fitur mulai dari berbagai informasi mengenai tempat wisata yang disediakan melalui artikel, memberikan komentar pada artikel yang tersedia, membeli tiket wisata secara online, dan juga dapat menambahkan tempat wisata yang ingin anda kunjungi kedalam daftar wisata anda.
